## 🚀 Features
✅ Responsive Navbar with smooth scroll effect  
✅ Multiple sections – **Home**, **About**, **Projects**, **Resume**  
✅ Modern UI built with **React-Bootstrap**  
✅ Easy to customize and deploy  
✅ Mobile-friendly design
